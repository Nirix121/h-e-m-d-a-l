<h1>MODoS</h1>
<h3>Linux/Windows/Termux</h3>

**How to install:**

**$** git clone https://github.com/Pix-head/MODoS.git<br>
**$** cd MODoS/<br>
**$** pip install -r requirements.txt<br>
<br>
**Interactive mode:**

**$** python modos.py<br>

OR

**$** python modos.py [HOST] [PORT] [THREADS] [DURATION]<br>
EXAMPLE: python modos.py http://google.com 80 50 3
